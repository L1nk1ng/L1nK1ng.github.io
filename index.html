Progetti di sicurezza informatica

Progetti

  • Analisi delle Vulnerabilità di un Sito Web

    Realizzazione di uno strumento di analisi delle vulnerabilità per siti web. Il progetto coinvolge la scansione automatica di un sito alla ricerca di possibili falle di sicurezza come XSS, SQL injection, e altro ancora. L'applicazione fornisce un report dettagliato delle vulnerabilità individuate.

    Link al progetto su GitHub

  • Sistema di Rilevamento delle Intrusioni (IDS)

    Implementazione di un sistema di rilevamento delle intrusioni per monitorare e identificare comportamenti sospetti nella rete. Il progetto utilizza tecniche avanzate di analisi del traffico e algoritmi di machine learning per riconoscere pattern di attacchi. Include una dashboard per la visualizzazione delle attività rilevate.

    Link al progetto su GitHub
